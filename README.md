## CV Builder React App
 - Creates a PDF file after submitting a form.
 - There a few options for CV design.
 - The required includes these categories: Basic Information, Education, Work Experience, and Skills.

## Framework used
- React(hooks)

## Library Used
- react-pdf/renderer
